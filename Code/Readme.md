# Code

In deze map staan alle afgeronde codes voor de werkende onderdelen van de scope.

* [main.py](https://github.com/Ivobr/Stage-4More/blob/50%25OpleverSet/Code/main.py) dit is de code waarin alles samen
  komt. De werking van de IO (Sensor) en het autonoom opakken en weg leggen van de bloem en de security settings.
* [IO.py](https://github.com/Ivobr/Stage-4More/blob/50%25OpleverSet/Code/IO.py) dit is de code waarin de werking van de
  IO (knop) in is getest. Hier staat enkel de werking van de IO in.
* [SetDrag.py](https://github.com/Ivobr/Stage-4More/blob/50%25OpleverSet/Code/setDrag.py) in deze code wordt de robot in
  een drag-mode gezet. Via deze modus kunnen de huidige coordinate en as posities makkelijk worden overgenomen voor een
  code.
