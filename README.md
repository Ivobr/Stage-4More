# Stage 4MT 50% opleverset

Dit is de 50% opleverset van mijn stage voor 4More Technology.

## Inleiding

Voor deze opdracht wordt is er een robotarm uitgekozen en geimplementeerd voor het verplaatsen van bloemen.
Deze opdracht is gedaan voor 4More Technology, dit is een klein bedrijf wat tel- en sorteermachines maakt voor het
automatiseren van het verwerken van sierbloemen.
